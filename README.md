# Attic Greek Trainer

A deployable React + Vite web app for studying Attic Greek.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

1. Push this folder to a GitHub repo
2. Import the repo into Vercel
3. Framework preset: Vite
4. Build command: `npm run build`
5. Output directory: `dist`

## Notes

- Progress is saved in local storage
- Use **Export progress** in the app to copy saved data
- This scaffold uses lightweight local UI components for easier deployment
